== Description ==

Plugin replaces the standard time controls (24h) with am/pm time controls.

== Installation ==

To install the plugin

1. Upload `wp-ampm` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

To use the plugin

1. Go to post editor page where after full page loading you will see input for 12 hours and AM/PM selectbox.


== Frequently Asked Questions ==

Q: How can I get back 24hours time controls?
A: You should turn off the plugin for this.
